# Restemblance LLC Website

Official website for **Restemblance LLC**, providing web development, software development, and STEM education services.

This project is built using the **HTML5 UP "Forty" theme** and customized for Restemblance LLC.

---

# Project Overview

This website serves as the digital platform for:

- Web and software development services
- STEM education initiatives
- Subscription-based development plans
- Donations and mission support

The site includes legal compliance pages and subscription integration through PayPal.

---

# Technology Stack

- HTML5
- CSS3
- JavaScript
- HTML5 UP Forty Theme
- PayPal Smart Payment Buttons
- Responsive grid layout

# Payment Integration

Subscriptions and payments are processed using **PayPal**.

Features include:

- Recurring subscription plans
- Secure payment processing
- PayPal hosted billing

PayPal SDK is loaded once per page and multiple subscription buttons are rendered dynamically.
